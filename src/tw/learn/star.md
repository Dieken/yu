## 字根小碼

星陳方案使用強音託，字根小碼 85% 以上情況下都是漢語拼音的第一個字母（yi 取 y，yu 取 v）。故而大部分的字根小碼不用特別記憶，可以猜出來。

由於很多字根拼音只有兩個字母，首字母容易被佔用，故而 15% 情況下會選取其後續字母（主元音）作為音託。只要特別記住以下字根即可：

| 小碼  | 字根         | 解釋             |
| :---- | :----------- | :--------------- |
| a     | `三八麻`       |                  |
| o     | `又尤由舟　艹` |                  |
| e     | `人文了業冊`   |                  |
| i　ㄧ | `七　心　臼`   |                  |
| i     | `矢世十　止自` |                  |
| u     | `不夂鬥`  |                  |
| r　ㄦ | `二兒而`       |                  |
| v     | `上小口`       | 讓出聲母給低頻根 |
